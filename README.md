QML::File version 0.02
================================

This module can be used to extract basic information from QML files, such as a list of properties, signals, functions, and child components.

INSTALLATION

To install this module type the following:

   perl Makefile.PL
   make
   make test
   make install

DEPENDENCIES

Test::More
